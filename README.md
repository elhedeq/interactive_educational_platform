# Interactive Educational Platform

A comprehensive Elearning platform built with Angular for front-end and PHP+MySQL for back-en. 
This application serves students, instructors, and administrators with dedicated dashboards and workflows.

You can see project documentation [Here!](https://docs.google.com/document/d/1IXFXvEIVWw357V_33gLlQJUGlTZ-LCtP/edit?usp=drive_link&ouid=114037078651867273295&rtpof=true&sd=true)

## Key Features

* **Public Portal**: Home page, course catalog, and detailed course previews.
* **Student Experience**:
    * Secure Login & Registration.
    * Course Checkout & Enrollment.
    * Learning Interface: Full-screen course viewer.
    * User Profile: Personal info management and enrolled course list.
* **Instructor Dashboard**:
    * Course Management: Create, edit, and view courses.
    * Grading System: Review and grade student quizzes and projects.
* **Admin Dashboard**:
    * System Overview & Stats.
    * Account Management.
    * Global Course Management.

## Tech Stack

* **Framework**: Angular 
* **Styling**: Bootstrap
* **Backend**: PHP
* **Database**: MySQL 

## Getting Started

1.  **Install Dependencies**
    ```bash
    npm install
    ```

2.  **Run Development Server**
    ```bash
    ng serve
    ```
    Navigate to `http://localhost:4200/`.

3.  **Build for Production**
    ```bash
    ng build
    ```
